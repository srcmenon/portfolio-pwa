Import CSV

📋 CSV format guide — click to expand

Your file must be a plain .csv with one row per purchase lot. Columns can be in either of these two orders:

#ColumnRequired?Example
1Dateoptional2024-03-15
2NamerequiredApple Inc
3TickerrequiredAAPL
4BrokeroptionalScalable
5TypeoptionalStock · ETF · MutualFund · Commodity
6Quantityrequired10
7Pricerequired178.50
8CurrencyoptionalEUR · USD · INR

Example row (with date):
2024-03-15,Apple Inc,AAPL,Scalable,Stock,10,178.50,EUR

Example row (without date — 7 columns):
Apple Inc,AAPL,Scalable,Stock,10,178.50,EUR

  • Each row = one purchase lot. Add multiple rows for the same stock if you bought at different times/prices.
  • Ticker must match Yahoo Finance (e.g. ^NSEI for Nifty, BTC-USD for Bitcoin).
  • Currency defaults to EUR if omitted. Prices are converted to EUR using live FX rates.
  • No header row needed — the app detects columns automatically.
  • Use the Export Portfolio as CSV button below to download a template from your existing data.
